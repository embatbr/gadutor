# gadutor

*Translate bolsonaristas' sentences to "mu lang".*

- twitter: @gadutor


## Running

### Manually

Fill the file *input.txt* in package *python*, do

```
$ cd python
$ python3 gadizator.py
```

and get the translation from file *output.txt*.
